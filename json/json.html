What is JSON

JSON (JavaScript Object Notation) is a lightweight standard text-based format for representing JavaScript Object syntax which makes it easy to send and receive data from client to server to be displayed on a webpage. Almost every major coding language has some functionality to read (parse) JSON which is simply converting JSON to a text format object or class that can be used inside a program.

JSON Data Types

JSON values cannot be one of the following data types:


a function
a date
underfined
comments are also not used
